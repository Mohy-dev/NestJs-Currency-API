## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# e2e tests
$ npm run test:e2e
```

## Database

```bash
# create database
$ npm run prisma:dev

# create database for testing
$ npm run prisma:test

# create database in container
$ npm run db:dev:restart
```

## Seed database

```bash
$ npm run prisma:seed
```
